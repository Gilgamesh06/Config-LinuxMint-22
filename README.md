# Script post instalacion Linux Mint 22

* **Instrucciones**

    * Clonar repo

    ```bash
        git clone https://github.com/Gilgamesh06/Config-LinuxMint-22.git
    ```

    * Dar permisos al script

    ```bash
        sudo chmod +x init.sh 
    ```
    * Ejecutar el script

    ```bash
        ./init.sh
    ```
